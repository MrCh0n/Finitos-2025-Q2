## Programas para Correr
### main.m
  Se puede resolver el ej1 con Mindlin y con 3D degenerado y distintos refinamientos\
  tiene 2 variables a cambiar
  - Type(1x1): tipo de elemento a usar
  - Div(1x1): cantindad de divisiones tanto en X como Y

  Cuando se cambia Type se tienen que comentar y descomentar las lineas
  - 66/67, distinto llamado a crearK
  - 146/147, distinto llamado para calcular esfuerzos

### main_plot.m
  Resuelve el ej1 para los refinamientos definidos y plotea los esfuerzos de 1 refinamiento especifico\
  se controla con 2 variables
  - Div(1x1): en que refinamiento plotear los esfuerzos
  - Divisiones(1x4): los distintos refinamientos que se calculan. Tiene que contener Div
